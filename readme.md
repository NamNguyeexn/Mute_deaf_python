# ** Hệ thống hỗ trợ giao tiếp người câm điếc **

## Công nghệ sử dụng :

- Sử dụng labelImg gán mẫu cho ảnh 
- Sử dụng datamodel của yolov5 hỗ trợ train ra model mới
